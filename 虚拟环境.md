## 虚拟环境

### Python

+ 安装

  ```python
  pip install virtualenv
  ```

+ 创建

  ```
  virtualenv -p /usr/bin/python3 venv(虚拟环境名称/路径)
  ```

+ 激活

  ````
  source ./venv/bin/activate
  ````

+ 退出

  ```
  deactivate
  ```

  

### Miniconda

+ 查看

  ```
  conda info --envs
  ```

+ 创建

  ```
  conda create --prefix=/opt/miniconda3/envs/venv python=3.7.3 -y
  ```

+ 激活

  ```
  conda activate venv
  ```

+ 退出

  ``` 
  conda deactivate
  ```
